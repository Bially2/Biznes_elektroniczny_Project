# JAK OTWORZYĆ PRESTASHOP

Aby uruchomić PrestaShop, otwórz konsolę **w folderze `prestashop`** i wpisz:

```bash
docker compose up -d
```
# Zatrzymywanie kontenerów
Aby zatrzymać kontenery bez ich usuwania:
```bash 
docker compose stop
```

Aby zatrzymać i usunąć kontenery (wolumeny pozostają):
```bash
docker compose down
```
Po odpaleniu Prestashop w przeglądarce wpisać: 
```bash 
http://localhost:8080
```

Aby wejść na Prestashop od strony admina: 
```bash
http://localhost:8080/admin248vwf6eo
```
